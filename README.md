# dockertron

![Dockertron promo](promo/dockertron.gif)

## Prerequisites
- [Docker Desktop](https://www.docker.com/products/docker-desktop)
- [Node](https://nodejs.org)
- [Yarn](https://classic.yarnpkg.com)

## Setup / Run

```
git clone https://github.com/jcs224/dockertron.git
cd dockertron
yarn install
yarn electron:serve
```
